# PRD: Natural Language Time Entry

## Problem
Freelancers waste time manually selecting client, project, and task when logging time. They think in phrases like "2 hours design for Acme" but have to click through 3-4 dropdowns.

## Solution
A single text input that uses AI to parse natural language into structured time entry data.

## User Stories
- As a freelancer, I want to type "2h design for Acme" and have it auto-create a time entry, so I save time on manual input
- As a freelancer, I want to see what the AI parsed before saving, so I can correct mistakes
- As a freelancer, I want the AI to suggest from my existing clients/projects, so it learns my workflow

## Functional Requirements
1. Text input field at the top of the time tracking page
2. AI parses input into: duration, client, project, description
3. Show parsed preview card below input
4. User confirms or edits before saving
5. Match client/project names against existing records (fuzzy match)
6. If no match found — suggest creating new client/project

## NOT in scope
- Voice input
- Calendar integration
- Batch entry from text

## Acceptance Criteria
- [ ] User types natural language → sees parsed preview in <2 seconds
- [ ] Correct parsing for formats: "2h", "2 hours", "30min", "1.5h"
- [ ] Matches existing clients with >80% accuracy
- [ ] Works in English, Ukrainian, Russian
- [ ] Mobile-friendly input
